# App Inventor 2 应用架构设计教程

> 覆盖：MVC架构、模块化、代码组织、设计模式

---

## 一、架构原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| 单一职责 | 每个组件只做一件事 |
| 模块化 | 功能模块解耦 |
| 可复用 | 通用代码抽离 |
| 可维护 | 结构清晰易改 |

---

## 二、MVC架构

### 2.1 Model（数据层）

```
// 数据模型
定义 创建用户模型(用户名, 邮箱)
  返回 创建字典(
    键值对("id", 获取毫秒数(现在)),
    键值对("username", 用户名),
    键值对("email", 邮箱),
    键值对("createdAt", 获取毫秒数(现在))
  )
```

### 2.2 View（视图层）

```
// 界面更新
定义 更新用户界面(用户数据)
  设置 标签_用户名.文本 = 获取键的值(用户数据, "username", "")
  设置 标签_邮箱.文本 = 获取键的值(用户数据, "email", "")
```

### 2.3 Controller（控制层）

```
// 用户交互处理
当 按钮_保存.被点击 时
  初始化局部变量 用户 = 调用 创建用户模型(
    文本输入框_用户名.文本,
    文本输入框_邮箱.文本
  )
  调用 保存用户(用户)
```

## 三、模块化设计

### 3.1 屏幕划分

| 屏幕 | 功能 |
|------|------|
| 启动屏幕 | 初始化、权限 |
| 登录/注册 | 认证 |
| 主屏幕 | 导航、功能入口 |
| 设置屏幕 | 配置 |
| 各功能屏幕 | 具体业务 |

### 3.2 模块抽离

```
// 通用模块：网络请求
定义 发起请求(URL, 方法, 回调)
  设置 Web客户端1.URL = URL
  调用 Web客户端1.发送GET请求()

// 通用模块：数据存储
定义 保存数据(键, 值)
  调用 TinyDB1.存储值(键, 值)

定义 读取数据(键, 默认值)
  返回 获取键的值(调用 TinyDB1.获取值(键, 默认值), "value", 默认值)
```

## 四、事件驱动

### 4.1 事件总线

```
初始化全局变量 事件队列 = 创建字典()

定义 发布事件(事件名, 数据)
  设置 全局变量 事件队列[事件名] = 数据

定义 订阅事件(事件名)
  返回 获取键的值(全局变量 事件队列, 事件名, 空文本)
```

### 4.2 事件处理

```
当 按钮_登录.被点击 时
  调用 发布事件("登录开始", 真)

当 全局变量 事件队列["登录开始"] = 真 时
  设置 标签_状态.文本 = "登录中..."
```

## 五、状态管理

### 5.1 应用状态

```
初始化全局变量 应用状态 = 创建字典()

定义 初始化状态()
  设置 全局变量 应用状态 = 创建字典(
    键值对("已登录", 假),
    键值对("当前用户", 空文本),
    键值对("主题", "light"),
    键值对("语言", "zh_CN")
  )

定义 设置状态(键, 值)
  设置 全局变量 应用状态[键] = 值

定义 获取状态(键)
  返回 获取键的值(全局变量 应用状态, 键, 空文本)
```

## 六、代码规范

### 6.1 命名规范

```
// 变量：驼峰命名
初始化局部变量 用户列表 = 创建列表()
初始化局部变量 当前页面 = 1

// 常量：全大写下划线
初始化全局变量 最大尝试次数 = 3
初始化全局变量 请求超时 = 30000

// 函数：动词开头
定义 保存数据() = ...
定义 加载配置() = ...
定义 验证输入() = ...
```

### 6.2 代码注释

```
// 模块功能说明
// 作者：xxx
// 日期：2024-01-01
// 更新：xxx

// 重要逻辑注释
// 这里处理了边界情况，防止崩溃
```

## 七、错误处理

### 7.1 统一错误处理

```
定义 处理错误(错误信息, 来源)
  调用 对话框1.显示消息对话框(
    "发生错误: " + 错误信息,
    "错误 - " + 来源,
    "确定"
  )
  // 记录日志
  调用 保存日志(来源, 错误信息)
```

### 7.2 容错机制

```
定义 安全获取(键, 默认值)
  初始化局部变量 值 = 获取键的值(全局变量 数据, 键, 空文本)
  如果 值 = 空文本 或 值 = 空 则
    返回 默认值
  返回 值
```

## 八、性能优化

### 8.1 减少全局变量

```
// ❌ 滥用全局变量
初始化全局变量 temp1 = ...
初始化全局变量 temp2 = ...

// ✅ 使用局部变量
当 按钮_点击.被点击 时
  初始化局部变量 temp = ...  // 用完释放
```

### 8.2 事件防抖

```
初始化全局变量 上次点击 = 0

当 按钮_防抖.被点击 时
  初始化局部变量 当前时间 = 获取毫秒数(现在)
  
  如果 当前时间 - 全局变量 上次点击 > 1000 则
    设置 全局变量 上次点击 = 当前时间
    调用 处理点击()
```
