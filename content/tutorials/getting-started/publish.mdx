| 选项 | 说明 |
|------|------|
| 调试APK | 未签名，可直接安装 |
| 发布APK | 已签名，需配置 |
| AAB | Google Play推荐格式 |

## 三、上架

### 3.1 应用市场

| 市场 | 特点 |
|------|------|
| Google Play | 全球，付费 |
| 应用宝 | 腾讯，国内 |
| 华为应用市场 | 华为手机 |
| 小米应用商店 | 小米手机 |

### 3.2 上架准备

- 应用图标（512x512）
- 截图（2-3张）
- 应用描述
- 隐私政策链接
